References + Materials

This page lists all the references and materials I have used in each section of my final project.

Modelling

  • City model - author: Kvngzalez, site: CGTrader
  • Cat model - author: googlefish, site: CGTrader
  • Blender - a free and open source 3D creation suite.

Spotlight

  • PBR Book, Section 12.3.1 (Spotlights) - my spotlight is implemented based off this.
  • Mitsuba 2, Section Emitters (Spotlight Source) - a retargetable forward and inverse renderer used for validation.

Layered BSDF

  • Material Test Ball - created by Benedikt Bitterli, the model used for testing my implemented BSDFs.
  • Weidlich and Wilkie - reference to the layered BSDF method I tried to implement (paper).
  • Generalization of Lambert's Reflectance Model - the Oren-Nayar model (paper).
  • PBR Book, Section 8.4.1 (Oren-Nayar Diffuse Reflection) - I implemented my rough diffuse surface based on this.
  • PBR Book, Section 8.4.4 (The Torrance-Sparrow Model) - referenced to implement my Torrance-Sparrow model.
  • PBR Book, Section 9.2.2 (Plastic Material) - the model I wanted to implement with thin dielectric over diffuse.
  • PBR Book, Section 8.5 (Fresnel Incidence Effects) - to implement the Fresnel Blend.
  • Mitsuba Smooth Plasic Material - used for validation.
  • Mitsuba Rouch Conductor Material - used for validation.
  • Mitsuba Smooth Diffuse Material - used for validation.

Fur Rendering

  • Marschner, Light Scattering from Human Hair Fibers - reference paper.
  • d'Eon11, An Energy-Conserving Hair Reflectance Model - reference paper.
  • d'Eon13, Importance Sampling for Physically-Based Hair Fiber Models - reference paper.
  • PBR Book, Section 3.7 (Curves) - referenced for implementation of curve primitives.
  • Matt Pharr, The Implementation of a Hair Scattering Model -reference material.
